# Day 1 - Orientation (Data Engineering Bootcamp)

## What is Data Engineering?

Data Engineering is the process of collecting, storing, transforming, and preparing data for analysis and machine learning.

## What Data Engineers do:

- Build data pipelines
- Clean and transform raw data
- Manage databases and data warehouses
- Work with big data tools

## Real-world Data Pipeline:

Data Source → Ingestion → Storage → Processing → Analytics

## Key Tools:

- Python
- SQL
- ETL tools
- Cloud platforms (AWS, Azure, GCP)
- Kafka, Spark, Airflow
- ETL (Extract → Transform → Load)
- ELT (Extract → Load → Transform)

## Goal of this Bootcamp:

To build real-world Data Engineering skills using hands-on projects.
