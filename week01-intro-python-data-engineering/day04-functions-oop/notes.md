# Day 4 - Functions and OOP Introduction

## Functions

Functions are reusable blocks of code.

```python
def greet(name):
    return "Hello " + name

print(greet("Saroj"))
Function with parameters
def add(a, b):
    return a + b
OOP (Object-Oriented Programming)

OOP is a way of structuring code using classes and objects.

Class Example:
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def display(self):
        print(self.name, self.age)

s1 = Student("Saroj", 24)
s1.display()

    ## Key Concepts
- Class
- Object
- Method
- Constructor (init)
```
